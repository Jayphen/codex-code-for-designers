# Claude Code for Designers

A free course that teaches designers to ship real products using Claude Code. No coding required.

**What you'll build:** A weather widget with live data, your Figma design, deployed to a real URL.

## Get started

1. Complete the [setup guide](https://weship.design/setup)
2. Paste this into your terminal:

```bash
mkdir weather-widget && cd weather-widget && curl -O https://raw.githubusercontent.com/janhaaland/claude-code-for-designers/main/CLAUDE.md && claude
```

3. Type: "Let's start Module 1."

## Course modules

1. **Your First Conversation** — Scaffold a project without writing code
2. **Show Claude Your Design** — Drop your Figma screenshot into the chat
3. **Write the PRD** — Create a product brief with Claude
4. **Build It** — Live weather data, your design, running locally
5. **Put It on the Internet** — Deploy to Vercel, get a real URL
6. **Ship and Share** — Polish, share, and celebrate
